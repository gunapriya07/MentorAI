# MentorAI

MentorAI is a smart study assistant powered by Google Gemini LLM, designed to help students explain concepts, summarize notes, and generate quizzes—all through a modern Streamlit interface.

# Features

**Explain Mode:** Breaks down complex topics into simple explanations with examples and real-world uses.

**Summarize Mode:**  Summarizes content into clear, concise bullet points.

**Quiz Mode:**  Generates quiz questions for self-assessment.

**Session Context:**  Remembers your session for seamless follow-up questions.

**Modern UI:**  Clean, responsive interface built with Streamlit.
